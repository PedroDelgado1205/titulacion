Bienvenido a SketchGallery

Aquí podrás crear tus obras maestras artísticas y organizarlas a tu gusto.

Tendrás infinitas formas de presentar tu galería a otros usuarios.

Características

Dibujo

En esta página podrás dar rienda suelta a tu creatividad. Dispones de tres secciones clave:

Capas

  • Puedes añadir o eliminar capas según lo requiera tu proyecto.
  • Reorganiza las capas con una interfaz de arrastrar y soltar.
  • Guarda tus dibujos, eligiendo un nombre y descripción, y también podrás descargarlos en tu dispositivo.

Lienzo

  • Tendrás un espacio gráfico compatible con tabletas digitalizadoras y dispositivos táctiles.

Herramientas

  • Puedes cambiar entre diferentes herramientas de dibujo, como: Bolígrafo, Pincel, Lápiz, Aerosol y Marcador.
  • Dispones de un borrador para correcciones.
  • Puedes cambiar el color y la opacidad de las líneas de cualquier herramienta.
  • El grosor de las líneas también es ajustable para las herramientas de dibujo y el borrador.

Galería

Aquí encontrarás todas tus obras creadas. Podrás verlas, cambiarles el nombre o descripción, editarlas o eliminarlas.

Reorganiza tus obras con una interfaz de arrastrar y soltar.

Perfil

Al registrarte, deberás dirigirte a esta sección para actualizar tu información y foto de perfil.

Podrás editar tu información cuantas veces quieras.

Barra de búsqueda

Busca a otros usuarios ingresando su nombre o alguna semejanza. Si están registrados, podrás ver sus galerías.

También podras ver sus perfiles.

Desarrollo del Proyecto

¿Por qué?

SketchGallery se creó para combinar la creación de dibujos y la gestión de los mismos en un solo lugar, optimizando el flujo de trabajo.

¿Quién?

Desarrollado por Pedro Delgado usando Django para el backend y HTML, CSS, JS y Bootstrap para el frontend. Los datos se almacenan en SQLite.

Puedes ver el repositorio en GitHub aquí, o explorar otros proyectos aquí.


Esta aplicación web fue desarrollada como un Proyecto Integrador en la carrera de Desarrollo de Software del Instituto Wissen.
